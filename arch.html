archie install thing


download arch iso

get rufus && flash the usb

reboot into the usb

load your preffered keymap with

loadkeys keymap

setfont ter-132b to make your font bigger

check your interface status with ip a or ip link (were gonna be using ethernet for the installation process) if at least one is up, you're okay to go

use ping google.com (or any other address) to check your internet connection. if you get replies youre good to go.
then use ctrl + c to stop the pinging

run timedatectl to update the system clock

use fdisk -l to indentify all your discs
pick the disc you want to install the system on. (we will refer to this disc as discX, where X represents the partition. yours might look like sdX (X being a letter representing the partition) or nvmeAnBpX (A & B being constant numbers, and X a number representing the partition)

once you've identified your disc use cfdisk /dev/disc to open the partitioning tool.
my reccomended partition layout is:
  - ~1 GB EFI partition
  - ~1/2 Of your RAM GB swap partition
  - Rest of the disc root partition

when you've set up your partitions choose save, then exit

format your partitions with the following commands:
mkfs.filesystem /dev/discC (for example ext4 or btrfs) (C being our root partition)
mkswap /dev/discB (B being our swap partition)
mkfs.fat -F 32 /dev/discA (A being our EFI partition)

let's mount them now!
mount /dev/discC /mnt to mount the root partition
mount --mkdir /dev/discA /mnt/boot the --mkswap flag is used here to create the /boot folder
& swapon /dev/discB to turn swap on

install the linux kernel along some necessary packages
pacstrap -K /mnt base linux linux-firmware i'd also add nano sudo networkmanager
press enter and wait until it finishes

genfstab -U /mnt >> /mnt/etc/fstab to generate the fstab file

change root to /mnt with arch-chroot /mnt

use ln -sf /usr/share/zoneinfo/Area/Location /etc/localtime to set your timezone (your path can be for example /Asia/Tokyo)
then use hwclock --systohc to update your hardware clock

now a quick overview on nano
nano is a text editor. the editing is really intuitive, but theres a couple of things you should remember:
  - press ctrl + o then enter to save a file
  - press ctrl + x to exit out of nano (i reccomended using clear after that)

nano /etc/locale.gen to open the locale gen file. uncomment (remove the # from the start of the line) the line with en_US.UTF-8 or en_GB.UTF-8 and any others you might want

then locale-gen to generate the locale(s)

edit ( nano /etc/locale.conf ) the locale configuration file with nano accordingly:
LANG=locale, locale being one of the locales you installed in step e2

edit /etc/hostname with nano accordingly:
hostname, hostname being any chosen hostname for you computer

if you changed the keymap, run mkinitcpio -P

set your root password with passwd
(this isn't your user password, this is something like an administrator password)
